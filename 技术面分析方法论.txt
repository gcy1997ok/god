2. 最实用的 RSI 技巧：背离（比超买超卖更精准，规避 “假拐点”）
背离是股价和 RSI 走势相反，代表趋势即将反转，是你操作超跌短线题材（如春节前的影视传媒）的核心找买点技巧，也是中线高抛的关键信号：
底背离（买）：股价创出新低 → RSI 未创出新低（甚至走高）→ 下跌无力，反转在即；
✅ 适配场景：你埋伏商业航天（Q2 初低吸）、影视传媒（春节前超跌），用底背离找精准买点，比单纯看 RSI<20 胜率高 80%；
顶背离（卖）：股价创出新高 → RSI 未创出新高（甚至走低）→ 上涨无力，回调在即；
✅ 适配场景：AI 算力龙头波段冲高、商业航天事件驱动涨停后，用顶背离判断止盈点，避免卖飞或被套。
补充：底背离需配合成交量放大（资金进场）+关键支撑位（如前低、均线）不破，胜率显著提升”，


KDJ 指标
K/D/J < 20	超卖区，下跌趋势尾声	低吸机会，优先等金叉确认
20 < K/D/J < 80	震荡区，趋势不明	
K/D/J > 80	超买区，上涨趋势尾声	止盈 / 减仓机会，优先等死叉确认
2. 核心买卖信号（金叉买，死叉卖，分强弱）
KDJ 的金叉 / 死叉是直接的操作指令，分低位金叉（强信号）、高位死叉（强信号）、中位金叉 / 死叉（弱信号，谨慎）：
金叉（买）：随后 K 线上穿 D 线 → 趋势由跌转涨；
✅ 强信号：金叉出现在KDJ<20 的超卖区（如影视传媒超跌后），胜率超 90%；
❌ 弱信号：金叉出现在 50-80 的中位区，易出现 “假金叉”，大概率是震荡，不操作；
死叉（卖）：随后 K 线下穿 D 线 → 趋势由涨转跌；
✅ 强信号：死叉出现在KDJ>80 的超买区（如商业航天涨停后、AI 算力冲高后），果断止盈；
❌ 弱信号：死叉出现在 20-50 的中位区，小概率可能是中线调整，稳妥起见建议抛出。
补充1：超买超卖不代表立刻反转：单边牛市中，RSI>80、KDJ>80 可能持续很久（比如 AI 应用龙头的主升浪），此时趋势为王，指标为辅，不要盲目高抛；单边熊市中，超卖也会持续，不要过早抄底；
忽略成交量：指标共振必须配合成交量—— 买入共振时，需放量上涨（代表资金进场）；卖出共振时，需放量下跌（代表资金出逃），缩量的共振信号大概率是假信号；


MACD 通用核心参数（默认即可，无需修改）
DIF 线：快速线，趋势拐点先行指标；
DEA 线：慢速线，趋势确认指标；
红绿柱（BAR）：多空动能强弱，红柱 = 多头占优，绿柱 = 空头占优，柱线长短 = 动能大小；
零轴：多空分水岭，零轴上 = 中期多头趋势，零轴下 = 中期空头趋势（中线操作的核心判断依据）。
二、MACD 核心实操信号（4 个核心，从基础到高阶，直接对应操作）
MACD 的信号优先级：零轴位置 > 背离 > 金叉 / 死叉 > 红绿柱变化，越往后越基础，结合使用才精准，以下信号均搭配操作策略，贴合你的板块需求。
1. 基础信号：金叉买，死叉卖（分强弱，避免假信号）
金叉 / 死叉是最直观的信号，但零轴上 / 下的金叉死叉，强度天差地别，单边趋势中零轴中间的金叉死叉大概率是震荡，切勿操作：
黄金金叉（强买入）：DIF在零轴上方上穿DEA→ 中期多头趋势中，回调后的二次启动，胜率超 85%
✅ 适配：AI 算力 / 应用中线波段，回调后的加仓 / 建仓点；
弱势金叉（弱买入，谨慎）：DIF 在零轴下上穿 DEA→ 空头趋势中的反弹，空间有限，仅适合短线题材（影视 / 商业航天）的超跌埋伏，不可中线持有；
死亡死叉（强卖出）：DIF 线上穿 DEA 线后，在零轴下再次下穿 DEA 线→ 中期空头趋势中，反弹后的二次下跌，果断止损 / 止盈；
弱势死叉（弱卖出，观望）：DIF 在零轴上下穿 DEA→ 多头趋势中的回调，中线标的（AI 应用）可持有，短线题材（影视）需减仓。
2. 动能信号：红绿柱的「缩脚 / 拐头 / 放大」（预判趋势拐点）
柱线变化比金叉死叉提前 1-2 个交易日发出信号，适合短线择时（影视传媒快进快出、商业航天博弈），是超短操作的关键：
红柱放大：多头动能增强，趋势延续，持有为主；
红柱缩量（拐头向下）：多头动能衰减，即将回调，短线止盈，中线减仓；
绿柱放大：空头动能增强，趋势延续，观望 / 止损；
绿柱缩脚（拐头向上）：空头动能衰减，即将反弹，短线低吸埋伏，中线轻仓试错。
3. 趋势信号：零轴的「突破 / 回踩」（中线趋势的核心依据）
零轴是 MACD 的「生命线」，突破零轴 = 趋势反转，回踩零轴 = 趋势确认，是你布局AI 算力 / 应用中线的核心判断标准，比任何短期信号都重要：
零轴突破（买）：DIF/DEA 从零轴下站上零轴，且红柱同步放大→ 中期趋势由空转多，中线可分批建仓（AI 应用 Q1 埋伏的核心信号）；
零轴回踩（买）：DIF/DEA 站稳零轴后，小幅回踩零轴不破位，且绿柱快速缩脚→ 多头趋势中的洗盘，中线加仓点（AI 算力波段低吸）；
零轴跌破（卖）：DIF/DEA 从零轴上跌穿零轴，且绿柱同步放大→ 中期趋势由多转空，所有板块果断清仓，不犹豫。
4. 高阶信号：MACD 背离（比 RSI 背离更稳，趋势拐点精准判断）
MACD 背离是趋势反转的最强信号，比 RSI 背离更贴合中期趋势，适合判断「短线题材的超跌底、中线标的的趋势顶」，分为顶背离（卖）、底背离（买），核心看股价和 MACD 柱线 / DIF 线的反向走势：
底背离（强买入）：股价创出新低 → MACD 柱线不创新低（绿柱缩脚）/DIF 线不创新低 → 空头动能耗尽，反转在即
✅ 适配：影视传媒节前超跌埋伏、商业航天 Q2 初回调低吸、AI 算力波段底部建仓；
顶背离（强卖出）：股价创出新高 → MACD 柱线不创新高（红柱缩量）/DIF 线不创新高 → 多头动能耗尽，回调在即
✅ 适配：影视传媒节后冲高止盈、AI 算力波段冲高减仓、商业航天事件驱动后的清仓。
补充1：零轴上金叉需结合“红柱是否持续放大”+“股价是否突破前高”，若红柱缩量或股价滞涨，需警惕假突破。

一、均线核心本质（先懂这个，才不会乱看）
 
均线 = 市场平均持仓成本线
- 股价在均线上方：多数人赚钱 → 多头占优
- 股价在均线下方：多数人被套 → 空头占优
- 均线方向：代表趋势方向
 
二、均线分析 5 大核心逻辑
1. 看趋势：先定多空（第一步）
- 多头排列（上涨趋势）
短期均线(5/10日) 在 中期(20/30日) 在 长期(60日) 之上，从上到下依次向下
逻辑：资金持续买入，成本不断抬高，趋势健康。


- 空头排列（下跌趋势）
长期均线 在 中期 在 短期 之上，且均线向上发散
逻辑：资金持续出逃，成本不断降低，反弹就是卖。
​
- 均线走平/缠绕（震荡趋势）
所有均线挤在一起 → 无趋势，高抛低吸，不追涨杀跌。
 
2. 看支撑与压力（最常用）
- 上涨中：均线=支撑
回踩均线不破 → 买点；
跌破均线 → 支撑变压力，趋势走弱。
- 下跌中：均线=压力
反弹到均线不过 → 卖点；
突破均线站稳 → 压力变支撑，趋势反转。
 
3. 看金叉 / 死叉（买卖信号）
- 金叉：短期均线上穿长期均线
例：5日穿10日、10日穿30日
逻辑：短期成本超过长期成本，资金开始主动买，看涨信号。
- 死叉：短期均线下穿长期均线
例：5日穿10日向下
逻辑：短期成本低于长期成本，资金开始逃，看跌信号。
 
⚠ 关键：
低位金叉才值钱，高位死叉最危险
（高位金叉多是骗线，低位死叉多是诱空）【在 A 股的主升浪中，高位顺向金叉才是最强的】
 
4. 看均线粘合 / 发散（判断变盘）
- 粘合：所有均线挤成一团
逻辑：多空平衡，即将变盘，等放量突破方向。
- 发散：均线间距越来越大
逻辑：趋势加速，顺势做，不要逆势抄底/逃顶。
补充1：“均线是趋势的‘滞后指标’，当趋势发生变化时，
需结合RSI/KDJ的背离信号（如股价创新高但均线未创新高）提前预判，避免依赖均线的交叉信号”。
补充2：短线题材股的核心是 5/10 日均线，20 日线是生命线，60 日线基本无参考意义；中线 AI 标的的核心是 20/30/60 日均线，5/10 日线仅用于短线择时。


K线形态分析：
长上影线/长下影线 往往预示局势反转 （T形态、锤子形态【出现在超跌区（RSI<20）更有效】）
乌云盖顶：T-1日阳线大幅上涨，T日高开低走，低位比T-1日实体红线1/2低
V型颈线位比较，或者W双底判断
反V、M顶，见顶信号
箱体震荡区间，观看是否突破支撑位/压力位
补充1：
上涨初期，放量长上影线是「仙人指路」，是洗盘信号，后续会继续上涨，绝非反转；
下跌初期，放量长下影线是「吊颈线」，是诱多信号，后续会继续下跌，绝非反转；
只有高位连续上涨后放量长上影线、低位连续下跌后放量长下影线，才是反转信号。
